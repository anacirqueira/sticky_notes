# Sticky_notes  ><

Projeto de notas adesivas criado para estudo da linguagem JavaScript

##

Link para voce testar => https://stickynotesss.netlify.app

##

 O que é possível fazer no Sticky notes? 

* Adicionar nova nota
* Editar uma nota
* Excluir (com dois cliques em cima da nota)
